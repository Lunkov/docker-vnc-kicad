# Docker container with VNC & KiCAD

Docker image is installed with the following components:
* Desktop environment Xfce4 or IceWM
* VNC-Server (default VNC port 5901)
* noVNC - HTML5 VNC client (default http port 6901)
* CADs:
    * KiCAD
    * FreeCAD
    * OpenROAD
* Browsers:
    * Mozilla Firefox
    * Chromium
